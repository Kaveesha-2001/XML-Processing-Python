# 🚗 Python XML Data Processing

This repository contains a practical implementation of **XML parsing and manipulation** using Python's built-in `xml.etree.ElementTree` (ET) library. This was developed as part of my data processing coursework to explore structured data handling.

---

## 🛠️ Key Features
The script demonstrates the following core XML operations:
*   **Safe Parsing:** Loads XML files with error handling for missing files.
*   **Traversal:** Iterates through child nodes to extract attributes and tags.
*   **Direct Access:** Accessing specific elements using index-based navigation.
*   **Search Operations:** 
    *   Using `.iter()` for deep searching specific tags.
    *   Using `.findall()` for structured data extraction.
*   **Data Modification:** Dynamically updating XML content and saving it to a new file (`output.xml`).
